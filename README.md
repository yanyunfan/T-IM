# T-IM

#### 介绍
基于t-io的一个极简聊天室应用。

t-io的测试性能，及其优势：
1. 2G内存，可支持30万人同时在线
2. 每秒支持1000万条聊天记录
3. 内置心跳超时检查、心跳定时发送能力
4. 掉线自动重连
5. 内置慢攻击防御机制，帮助应用自动拉黑嫌疑IP
6. 全方位的数据监控能力
