# T-IM

#### 介绍
基于t-io的一个极简聊天室应用。

t-io的测试性能：
1. 2G内存，可支持30万人同时在线
2. 每秒支持1000万条聊天记录

#### 软件架构
- 后端：spring boot 
- 前端：H5

#### 安装教程

1.  后端启动TImApplication
2.  前端访问：http://192.168.1.6/index.html
